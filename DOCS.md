# Description on the crawler data 
  
`category`. 

`forum`.  
Sub-forum name 

`thread_title`. 
  
`thread_url`. 

`username`.  
Name of the user that made the post 
  
`thread_url`.   
Url of the thread where the message is from 
  
`post_author_title`.  
Title of the user that made the post 
  
`post_id`.  
Id that the forum gave the post 
  
`timestamp`.  
server timestamp of when the message was posted 
  
`message_nr`.  
The number of the message from the thread. The first message in a thread starts with number 1. 
  
`post_message`.  
text content of the message post 

`quotes`.  
the list of post_id of the messages that the post referenced
